# US11 - Display Race Statistics

**Status:** ✅ Complete

As a race organizer, I want the system to display race statistics, so that I can quickly review participation breakdowns.

The statistics should include:
- Total number of males
- Total number of females
- Total number of males (U18)
- Total number of females (U18)
- Total number of males unaffiliated (excluding U18 males)
- Total number of females unaffiliated (excluding U18 females)

Unaffiliated means runners without a club.
